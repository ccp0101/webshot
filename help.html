This API creates a screenshot of a webpage.

A screenshot of New York Times homepage: /?url=http://www.nytimes.com

Parameters

  • width: Browser's viewport width in px. Default: 1200px
  • format: Output image format: png or jpg. Default: png
  • zoom: Browser's zoom level. Default: 1.
  • timeout: Page rendering timeout in miniseconds. Default: 30000.
  • delay: Wait before taking a screenshot. Default: 200.

Retina (2x) high-resolution screenshot: /?url=http://www.nytimes.com&zoom=2&width=2048px
